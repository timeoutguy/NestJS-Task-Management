
## Description

[Nest](https://github.com/nestjs/nest) task management.

## Running the app

Clone this repository and run

```bash
# instaling dependenciess
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
